# CoolQNetPlugin
酷Q非官方VB.NET SDK<br>

酷Q Air 非官方 .NET SDK，使用 VB 编写。该SDK可让开发人员使用托管代码库对酷Q Air 进行开发。

## SDK特性
当前暂无特性
### 即将到来
* 允许使用.NET语言生成的插件DLL直接被酷Q读取，而不用执行其它操作。
* 简洁，容易使用
